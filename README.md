# 📊 Sales & Revenue Growth Tracker (Excel Project)

## 🎯 Objective
To analyze sales performance, revenue, and growth trends using an interactive Excel dashboard.

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Dashboards  
- Basic Excel Formulas  

---

## 📂 Dataset Overview
Contains sales data including:
- Revenue, Profit, Units Sold  
- Region, Product, Sales Channel  
- Targets and Achievement %

---

## 📊 Key Features
- Revenue & profit tracking  
- Regional and product performance analysis  
- Monthly sales trend analysis  
- Target vs achievement comparison  
- Interactive dashboard with filters  

---

## 📈 Key KPIs
- Total Revenue  
- Total Profit  
- Units Sold  
- Achievement %  
- Growth Rate %  

---

## 📊 Dashboard Insights
- Identifies top-performing regions  
- Highlights best-selling products  
- Tracks sales growth trends  
- Evaluates target achievement
